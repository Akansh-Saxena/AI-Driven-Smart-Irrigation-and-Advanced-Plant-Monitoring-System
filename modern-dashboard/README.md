# Smart Irrigation - Modern Dashboard

This is a [Next.js](https://nextjs.org) web application that acts as the central command center for the Cyber-Physical Smart Irrigation System. 

## Features

- **Real-Time AI Analytics:** Monitor predictive Edge AI readings including soil moisture, weather data, and crop stress.
- **Interactive UI:** Next.js pages styled with modern UI principles (Glassmorphism, animations) and responsive mobile design.
- **AR / BCI Integration Ready:** Setup to receive and visualize high-speed metrics.
- **MQTT/WebSocket Streaming:** Enables real-time two-way communication between the ESP32 Core and the frontend interface.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server locally:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the live dashboard.

## Deployment

This dashboard is designed to be effortlessly deployed on the [Vercel Platform](https://vercel.com/new). The `main` branch can be automatically fetched and deployed, providing a free and robust hosting solution for your hardware interface.
